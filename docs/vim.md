# VIM

A ubiquitous text editor.

```bash
vi
```

## Modes

+ Command <esc>
+ Insert <i>
+ Last Line

## ShortCuts

+ <u> Undo
+ <ctrl> + <r> ReDo
+ dd : delete a line

## Search

+ /Hello + <enter> (first element with Hello)
+ <n> looks for (next element with Hello)
+ <shift> + <n> (searches the last element)
+ <ctrl> + <f> (page down)
+ <ctrl> + <b>

## Save and quit

+ wq
+ q!
